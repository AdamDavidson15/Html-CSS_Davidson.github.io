Identyfikatory ID

Pojęcie Znaczenie
Co znaczy skrót ID? Identyfikator elementu.
Jaki jest cel stosowania ID? Nadanie unikalnego stylu elementowi.
Jakie jest znaczenie znaku #? Służy do definiowania identyfikatora w CSS.
Jak następuje wywołanie ID? id="nazwa"
Uwagi o ID
  • ID powinno być unikalne.
  • ID może wystąpić raz.
  • ID ma większy priorytet niż klasa.
